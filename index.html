よくあるご質問

土日も診療していますか？

+

はい、土曜日は18:30まで、日曜日も18:00まで診療しています。平日お忙しい方もお気軽にお越しください。

当日予約は可能ですか？

+

空きがあれば当日予約も承ります。特に急患の場合は優先的に対応いたします。まずはお電話でご相談ください。

駐車場はありますか？

+

病院の正面に駐車場を完備しています。バリアフリー対応で、車椅子の方でも安心してお越しいただけます。

子どもを連れて行っても大丈夫ですか？

+

お子様連れでも安心です。スタッフがお子様の対応もいたします。小児歯科の詳細についてはこちらのページも参考にしてください

親知らずの抜歯は痛いですか？

+

口腔外科専門医が担当し、麻酔を使用しますし、必要に応じて笑気ガス鎮静も使用します。痛みを最小限に抑えた治療を心がけています。

インプラント治療の費用はどのくらいですか？

+

1本あたり35万円〜となりますが、インプラントはオーダーメイドの治療なので、本数や必要な検査、処置により症例により金額が大きく変わります。CT診断による詳細な治療計画をもとに、正確な費用をお見積もりいたします。詳細についてはこちらのページも参考にしてください

保険は使えますか？

+

一般的な虫歯治療や親知らず抜歯など基本的な治療は保険適用です。性能や美観なので自費の治療を選択することもできます。自費治療についても事前に詳しくご説明いたします。

他院で断られた難しい症例でも対応できますか？

+

口腔外科専門医が在籍しているため、一般歯科では対応困難な症例も治療可能です。入院や、大掛かりな手術が必要と認めた場合は、連携している医療機関に紹介いたします。まずはご相談ください。詳細についてはこちらのページも参考にしてください

感染対策はどうなっていますか？

+

クラスB滅菌装置を備えており、器具の滅菌はもちろん、空気清浄機の設置、定期的な換気など、徹底した感染対策を実施しています。

治療期間はどのくらいかかりますか？

+

症状や治療内容により異なります。初診時に詳細な治療計画と期間をご説明いたします。一般的な虫歯の治療期間についてはこちらのページも参考にしてください
